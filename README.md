# paybackboys.com

Official site of Payback Boys

## Install dependencies

```bash
$ npm i
```

## How to build and run

```
$ wrangler dev
```

## Deployment

Currently paybackboys.com is served with Cloudflare Workers. Deployment is automatically done with pushing commit to master branch.
